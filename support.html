    Where can I get support for Musify?

    You can get support for Musify in the Musify Support Discord server.

    What is the default prefix for Musify?

    The default prefix for Musify is >, but can be changed with the >prefix [new prefix], command. See Config for more information. At any time you can tag the bot to find its prefix in that guild.

    Is Bot Support Spotify?

    Yes, Bot Support Spotify, 24/7 and Autoplay.

    Where can I vote for Musify?

    You can vote for the Musify bot Top.gg.
